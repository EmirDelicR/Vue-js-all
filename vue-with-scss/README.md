# vue-with-scss

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Some CSS attributes

- letter-spacing: 0.3px;

- word-spacing: 0.3px;

- unicode-range: U+41-5A; (Use in font-face)

- position: static|absolute|relative|fixed|sticky;

(use position with top, left, right, bottom)
